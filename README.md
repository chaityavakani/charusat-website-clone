# CHARUSAT Website Clone

A responsive frontend clone of the CHARUSAT University website developed using HTML and CSS.

> This project was created for educational and learning purposes only. It is not affiliated with or endorsed by CHARUSAT University.

----
## Features

- Responsive Design
- Clean User Interface
- Navigation Bar
- Hero Section
- About Section
- Courses Section
- News & Events
- Footer
- Mobile Friendly
- Smooth Scrolling

---

## Technologies Used

- HTML5
- CSS3
